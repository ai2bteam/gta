# GTA 6 Asset Library — индекс

Обновляется Claude'ом при каждом изменении. Правила — в [CLAUDE.md](CLAUDE.md).

```
gta/
├── characters/    (пусто)
├── locations/     (пусто)
├── vehicles/      (пусто)
├── logos/         (пусто)
├── screenshots/   (пусто)
├── artwork/       (пусто)
├── maps/          (пусто)
├── ui/            (пусто)
└── misc/          (пусто)
```

## Как пользоваться

- Кидай новые файлы в любую папку (или прямо в корень) — Claude при следующей сессии разложит, переименует и опишет.
- Поиск: смотри теги в `*.md` рядом с файлами.
