# Источники материалов по GTA 6

Реестр проверенных сайтов для пополнения библиотеки. Обновляй при находке новых.

| Источник | Что даёт | Доступ | Заметки |
|---|---|---|---|
| Rockstar Games (пресс-кит) | официальные скриншоты, арты, трейлеры в 4K | архивы от пользователя | основа библиотеки: characters/, locations/, artwork/, screenshots/ |
| gtabase.com/gta-6/ | базы данных: персонажи, 200+ машин, карта и округа, оружие, банды | **только браузер** — curl/WebFetch блокирует Cloudflare; картинки вытаскивать через fetch внутри страницы → локальный HTTP-приёмник | источник maps/, CHARACTERS.md, LEONIDA.md, VEHICLES.md |
| rockstarintel.com | новости индустрии: продажи, маркетинг, суды, релизные детали | WebFetch работает | для новостных дайджестов (misc/NEWS_DIGEST_*.md) |
| playground.ru/gta_6 | новости на русском, галереи (112 офиц. + 258 фанатских) | WebFetch работает | галерея офиц. скриншотов дублирует пресс-кит; фанатскую не берём (авторские права) |

## Правила отбора

1. **Не дублировать**: перед скачиванием проверь INDEX.md — пресс-кит у нас уже в 4K, пережатые копии с сайтов не нужны.
2. **Фанатский контент** (карты, арты) — только с указанием автора в sidecar (`license: Fan-made / community`).
3. **Тексты сайтов не копировать** — факты пересказывать своими словами со ссылкой на источник.
4. **Официальный контент Rockstar** — `license: Rockstar promo / fair use`.
