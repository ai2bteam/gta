# artwork/ — индекс

Официальные арты GTA VI из пресс-кита Rockstar (пак обоев, июль 2026). Каждый сюжет лежит в своей подпапке и содержит один и тот же арт в кропах: `landscape` 3840x2160, `portrait` 2160x3840, `phone` 2160x4800, `tablet` 3840x3840, `square` 2160x2160, `ultrawide` 5120x2160 (исключения по разрешению отмечены ниже). У каждого файла есть sidecar `.md` с тегами.
